# 🌍 GlobeTap

A geography guessing game with a 3D globe. Test your world knowledge — tap where you think each location is!

## Features

- **3D Globe** — High-res NASA Blue Marble imagery, rotatable & zoomable
- **Daily Challenge** — Same 5 locations for everyone each day
- **Practice Mode** — Random locations, play unlimited
- **500+ Locations** — Spanning all continents, from capitals to hidden gems
- **Scoring** — Great-circle distance calculation, up to 1000 points per round
- **Statistics** — Performance history, continent accuracy, round analysis, best/worst guesses
- **Leaderboard** — Compete with friends

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- react-globe.gl (Three.js)
- Recharts
- React Router

## Getting Started

```bash
npm install
npm run dev
```

## How to Play

1. Sign up or log in
2. Choose Daily Challenge or Practice Mode
3. Each round shows a location name — tap the globe where you think it is
4. Score is based on distance: closer = more points (max 1000/round, 5000 total)
5. After guessing, see the arc from your guess to the actual location
6. Track your progress in Statistics
